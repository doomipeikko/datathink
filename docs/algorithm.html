The Algorithm

My algorithm is based on the Periodic Table of elements. I wanted to design an algorithm that asks a user to input an element, compares the input to a predetermined dataset and if a match is found, displays the stats of that element. The application will accept either the element's name or the element's symbol.

I also wanted the application to print an error message if a non-existing element was typed (or if the element's name was misspelled).

For easier use, I wanted the app to keep going until the user was satisfied and decided to end the application themselves by using the Quit command.

There's also a quick debugger that converts the input to the correct format (capitalized first letter) so JavaScript can understand the user input without immediately throwing an error message if the user types ca instead of Ca (for Calcium), for example.

The Pseudocode

    BEGIN application

    Request user input

    [User input]

        (Convert input to correct format)

    Assess user input

        IF input is "Quit" 
            THEN end program

        ELSE Validate input

    Compare input against dataset (retrieved from /data/elements.xml)

        IF element name is found

            THEN print element

        ELSE IF element symbol is found

            THEN print element

        ELSE print error message

    Loop back to BEGIN
