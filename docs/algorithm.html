The Algorithm

My algorithm is based on the Periodic Table of elements. I wanted to design an algorithm that asks a user to input an element, compares the input to a predetermined dataset and if a match is found, displays the stats of that element. I also wanted the program to print an error message if a non-existing element was asked for so I used an if-else comparison.

For easier use I wanted the app to keep going until the user was satisfied and decided to end the program themselves. For this, I programmed a while loop.

There's also a quick debugger that transforms the user input to lowercase so JavaScript and Python can understand the user input without immediately throwing an error message if the user types ca instead of Ca (for Calcium), for example.

The Pseudocode

    BEGIN

    User input

    Assess input

        IF "Quit" THEN end program

        ELSE Validate

    Validate input

        IF correct element

            THEN print element

        ELSE print error message

    Loop back to BEGIN
